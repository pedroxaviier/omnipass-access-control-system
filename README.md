# omnipass-access-control-system
Access control and audit system built with Node.js. Features real-time monitoring, login tracking, fraud detection, and immutable logs using SHA256 hash chaining. Includes admin dashboard, ticket system, and compliance-focused reporting (LGPD/ISO 27001).
